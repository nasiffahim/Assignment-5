Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an element in DOM, getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll(). In most cases we will use the getElementById() functionality but other cases also has some uses as well. The easiest way to find an HTML element in the DOM, is by using the element id. If the element is found, the method will return the element as an object (in element). If the element is not found, element will contain null.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Key differences between interHeml, innerText and textContent
1.innerHtml returns all text, including html tags, that is contained by an element.
2.innerText returns all text contained by an element and all its child elements.
3.textContents is all text contained by an element and all its children that are for formatting purposes only.

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript process where instead of adding separate event listeners to multiple child elements, a single listener is placed on their parent. This approach takes advantage of event bubbling, when an event occurs in the target element and it moves upward in its DOM tree. This method works well when there is a large number of dynamically created elements, as it enhances performance by requiring only one event listener rather than multiple individual ones.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the element that was clicked, it can be a child element or a div container and then travels up to its parent, grandparent, and so on, until it reaches the body tag of the html file. Inner (child) elements execute their event listeners first, followed by outer (parent) elements as the event propagates upward.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript we can create elements, add them to the DOM, and remove them as needed.
1. Create an element with document.createElement("tagName"), here tagName is the html tags such as p, h1, div etc.
2. Adding an element by using .appendChild() method, this adds the new element inside the parent element.
3. Removing an element by using .remove() or a specific child element then .removeChild() method, this removes an existing element from the parent element.
